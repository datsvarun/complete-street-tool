The aim was to have a web-application where user could just click on a few streets pulled from OSM within their neighbourhood, they want to impacted, the tool would then with the click of a button generated the necessary geospatial center-line graph, create intersections, generate the IRC Code compliant streets and will output the designed street plan which the user will be free to print, publish and take to their local representative/urban local body to lobby for. 

The current development proposes to start work as a QGIS Plugin. The geospatial capability and powerful vector asset editing position it as a formidable tool. With python scripting integrated it also enables iterations and GUI is taken care of allows me to focus on the scripting and logic of it.

The tool may later transition to a JS based Web-App with the underlying logic brought over to increase it's wider appeal as QGIS still proves to be a barrier for entry for a majority of the users.
